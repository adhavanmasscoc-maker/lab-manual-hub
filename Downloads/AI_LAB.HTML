EX NO: 1 - Implement Breadth First Search using Python

Date: ____________

Aim

To write a program to implement Breadth First Search (BFS) traversal algorithm on a graph using Python.

Algorithm

  1. Start BFS from the given node.
  2. Insert the starting node into a queue.
  3. Mark the node as visited.
  4. Remove a node from the queue and print it.
  5. Insert all unvisited adjacent nodes into the queue.
  6. Repeat until the queue becomes empty.

Program

from collections import deque
def bfs(graph, start):
    visited = []
    queue = deque([start])
    while queue:
        node = queue.popleft()
        if node not in visited:
            print(node, end=" ")
            visited.append(node)
            queue.extend(graph[node])
graph = { 'A': ['B', 'C'], 'B': ['D'], 'C': ['E'], 'D': [], 'E': [] }
print("BFS Traversal:")
bfs(graph, 'A')

Output

BFS Traversal: A B C D E

Result

The program to implement Breadth First Search was executed successfully.

EX NO: 2 - Implement Depth First Search using Python

Date: ____________

Aim

To write a program to implement Depth First Search (DFS) traversal algorithm on a graph using Python.

Algorithm: DFS using Recursion

  1. Start DFS from the given node.
  2. If the node is not visited: Print the node, Mark it as visited.
  3. For each adjacent node: Call DFS recursively.
  4. Continue until all nodes are visited.

Program

def dfs(graph, start, visited=[]):
    if start not in visited:
        print(start, end=" ")
        visited.append(start)
        for node in graph[start]:
            dfs(graph, node, visited)
graph = { 'A': ['B', 'C'], 'B': ['D'], 'C': ['E'], 'D': [], 'E': [] }
dfs(graph, 'A')

Output

A B D C E

Result

The program to implement Depth First Search was executed successfully.

EX NO: 3 - Implement Alpha-Beta Pruning using Python

Date: ____________

Aim

To write a program to implement Alpha-Beta Pruning algorithm for game tree optimization using Python.

Algorithm

  1. Start from the root node (MAX player).
  2. Initialize: Alpha = −∞ (best value for MAX), Beta = +∞ (best value for MIN).
  3. Recursively evaluate child nodes.
  4. Update alpha and beta values. Prune remaining branches when alpha ≥ beta.
  5. Return the optimal value.

Program

import math
def alphabeta(depth, nodeIndex, maximizingPlayer, values, alpha, beta):
    if depth == 3:
        return values[nodeIndex]
    if maximizingPlayer:
        best = -math.inf
        for i in range(2):
            val = alphabeta(depth+1, nodeIndex*2+i, False, values, alpha, beta)
            best = max(best, val)
            alpha = max(alpha, best)
            if beta <= alpha:
                break # pruning
        return best
    else:
        best = math.inf
        for i in range(2):
            val = alphabeta(depth+1, nodeIndex*2+i, True, values, alpha, beta)
            best = min(best, val)
            beta = min(beta, best)
            if beta <= alpha:
                break # pruning
        return best

values = [3, 5, 6, 9, 1, 2, 0, -1]
print("Optimal value:", alphabeta(0, 0, True, values, -math.inf, math.inf))

Output

Optimal value: 5

Result

The program to implement Alpha-Beta Pruning was executed successfully.

def fuzzy_control(temp):
    # Fuzzification
    if temp < 20:
        cold = 1; warm = 0; hot = 0
    elif temp < 30:
        cold = 0; warm = 1; hot = 0
    else:
        cold = 0; warm = 0; hot = 1
        
    # Rule base
    fan_speed = (cold * 1) + (warm * 2) + (hot * 3)
    return fan_speed

temperature = 28
print("Fan Speed Level:", fuzzy_control(temperature))

EX NO: 5 - Tokenization, Stopword Removal, and Stemming using NLTK

Date: ____________

Aim

To write a program to perform tokenization by word and sentence, eliminate stopwords, and perform stemming operations using NLTK library in Python.

Algorithm

  1. Input a sentence.
  2. Split the sentence into individual words (tokens).
  3. Load English stopwords.
  4. Remove common words like is, the, and, in.
  5. Reduce words to their root form using a stemmer.
  6. Display the final processed words.

Program

import nltk
from nltk.tokenize import word_tokenize
from nltk.corpus import stopwords
from nltk.stem import PorterStemmer

# Download required NLTK data
nltk.download('punkt')
nltk.download('stopwords')
nltk.download('punkt_tab')

text = "Artificial intelligence is changing the world"

# Tokenization
tokens = word_tokenize(text)

# Stopword removal
stop_words = set(stopwords.words('english'))
filtered_words = [w for w in tokens if w.lower() not in stop_words]

# Stemming
ps = PorterStemmer()
stemmed_words = [ps.stem(w) for w in filtered_words]

print("Tokens:", tokens)
print("After Stopword Removal:", filtered_words)
print("After Stemming:", stemmed_words)

Output

Tokens: ['Artificial', 'intelligence', 'is', 'changing', 'the', 'world']
After Stopword Removal: ['Artificial', 'intelligence', 'changing', 'world']
After Stemming: ['artifici', 'intellig', 'chang', 'world']

Result

The program to perform tokenization, stopword removal and stemming was executed successfully.

EX NO: 11 - Object Detection and Tracking using Kalman Filter, Camshift

Date: ____________

Aim

To detect a moving object in a video and highlight it using a bounding box.

Algorithm

  1. Convert frames to grayscale
  2. Compute frame difference
  3. Remove noise using threshold
  4. Draw bounding box on moving object
  5. Save output video

Program

import imageio, numpy as np
r = imageio.get_reader('/content/slow_traffic_small.mp4')
w = imageio.get_writer('/content/output_box.mp4', fps=10)

prev = np.mean(r.get_data(0), axis=2)
for i in range(1, 30):
    cur = r.get_data(i)
    gray = np.mean(cur, axis=2)
    diff = np.abs(gray - prev)
    y, x = np.where(diff > 40) # higher threshold
    if len(x) > 500: # ignore small noise
        y1, y2, x1, x2 = y.min(), y.max(), x.min(), x.max()
        cur[y1:y2, x1:x2:5] = [0,255,0]
        cur[y1:y2:5, x1:x2] = [0,255,0]
        cur[(y1+y2)//2, (x1+x2)//2] = [255,0,0]
    w.append_data(cur)
    prev = gray
w.close()
print("Saved output_box.mp4")

Output

Saved output_box.mp4

Result

The Object Detection and Tracking program was executed successfully.
