EX NO: 1 - Half Pyramid of stars using loop structures

AIM

To write and execute a Python program to print a half pyramid pattern of stars (*) using loop structures.

ALGORITHM

  1. Read the number of rows (n).
  2. Use an outer loop to control rows (1 to n).
  3. Use an inner loop to print stars for each row.
  4. Print a newline after each row iteration.

PROGRAM

# Half Pyramid of Stars using for loop
n = int(input("Enter number of rows: "))
for i in range(1, n + 1):
    for j in range(i):
        print("*", end=" ")
    print()

  2. Remove negative and zero values using a loop.
  3. Generate time sequences and calculate average power.

power = [120, 150, -20, 180, 0, 210, 250, -15]

    if value > 0:
        clean_power.append(value)

time = []

    time.append(count)
    count = count + 1

total = 0
for value in clean_power:
    total = total + value

print("Average Power:", average)

plt.plot(time, clean_power)
plt.xlabel("Time")
plt.ylabel("Power")
plt.title("Power Consumption")

import numpy as np

temperature_data = np.array([65, 70, 75, 80, 85, 90])

    print("WARNING: HIGH TEMPERATURE")

    print("WARNING: HIGH VIBRATION")

    print("MACHINE STATUS: NORMAL")

EX NO: 7 - Load forecasting with ANN

AIM

To predict future electricity demand using simple Artificial Neural Networks (ANN) propagation logic.

ALGORITHM

  1. Normalize historical load data to a 0-1 scale.
  2. Initialize weights, biases, and learning rate.
  3. Iterate through epochs calculating forward propagation output and error.
  4. Update weights and biases using backpropagation.
  5. Predict future load utilizing optimized parameters.

PROGRAM

import numpy as np

load_data = np.array([100, 120, 140, 160, 180])
print("LOAD FORECASTING USING ANN")
print("Historical Load Data:", load_data)

max_load = np.max(load_data)
normalized_load = load_data / max_load
print("\nNormalized Load Data:", normalized_load)

weight = 0.5
bias = 0.2
learning_rate = 0.1
epochs = 100

for epoch in range(epochs):
    for i in range(len(normalized_load) - 1):
        input_value = normalized_load[i]
        target_value = normalized_load[i+1]
        
        # Forward propagation
        output = (input_value * weight) + bias
        error = target_value - output
        
        # Backpropagation (Weight Update)
        weight = weight + learning_rate * error * input_value
        bias = bias + learning_rate * error

future_input = normalized_load[-1]
predicted_normalized_load = (future_input * weight) + bias
predicted_load = predicted_normalized_load * max_load

print("\nFinal ANN Parameters:")
print("Weight:", round(weight, 3))
print("Bias:", round(bias, 3))

print("\nPredicted Future Load:")
print(round(predicted_load, 2), "MW")
print("\nLoad Forecasting Completed Successfully")

OUTPUT

LOAD FORECASTING USING ANN
Historical Load Data: [100 120 140 160 180]
Normalized Load Data: [0.555 0.666 0.777 0.888 1.]
...
Predicted Future Load:
200.0 MW
Load Forecasting Completed Successfully

RESULT

The Artificial Neural Network successfully predicted future electrical load based on historical data.

EX NO: 8 - Power quality signal classification

AIM

To classify power quality disturbances such as normal, sag, and swell using SVM machine learning.

ALGORITHM

  1. Create arrays for training voltage features and corresponding state labels (0=Sag, 1=Normal, 2=Swell).
  2. Initialize an `SVC` (Support Vector Classifier) with a linear kernel.
  3. Fit the model to the training data.
  4. Predict categories for new test voltage samples and map them to readable text.

PROGRAM

import numpy as np
from sklearn import svm

print("POWER QUALITY SIGNAL CLASSIFICATION USING SVM")
voltage_signal = np.array([0.82, 0.85, 0.88, 0.95, 1.00, 1.02, 1.10, 1.15, 1.20])
print("Voltage Signal Samples:", voltage_signal)

X_train = np.array([
    [0.8], [0.85], [0.88], # Sag
    [0.95], [1.0], [1.05], # Normal
    [1.12], [1.15], [1.2]  # Swell
])
y_train = np.array([0, 0, 0, 1, 1, 1, 2, 2, 2])

model = svm.SVC(kernel='linear')
model.fit(X_train, y_train)

predictions = model.predict(voltage_signal.reshape(-1, 1))
label_map = {0: "Voltage Sag", 1: "Normal Voltage", 2: "Voltage Swell"}

print("\nClassification Results:")
for v, p in zip(voltage_signal, predictions):
    print(v, "→", label_map[p])

print("\nPower Quality Analysis Completed Successfully")

OUTPUT

POWER QUALITY SIGNAL CLASSIFICATION USING SVM
Voltage Signal Samples: [0.82 0.85 ... 1.2 ]
Classification Results:
0.82 → Voltage Sag
...
1.0 → Normal Voltage
...
1.2 → Voltage Swell

RESULT

The power quality classification system successfully identified voltage disturbances such as sag, normal, and swell.

EX NO: 9 - AI For Smart Energy Monitoring

AIM

To monitor daily energy consumption and intelligently detect abnormal usage using an MLP Classifier (Neural Network).

ALGORITHM

  1. Provide historical consumption data labeled as Low, Normal, and High.
  2. Train the `MLPClassifier` using the datasets.
  3. Predict the category for the latest energy usage input.
  4. Print contextual suggestions/alerts based on the classification result.

PROGRAM

import numpy as np
from sklearn.neural_network import MLPClassifier

print("AI BASED SMART ENERGY MONITORING SYSTEM")
# 0 = Low, 1 = Normal, 2 = High
X_train = np.array([
    [10], [12], [13],
    [14], [15], [16],
    [18], [20], [22]
])
y_train = np.array([0, 0, 0, 1, 1, 1, 2, 2, 2])

model = MLPClassifier(hidden_layer_sizes=(5,), max_iter=1000, random_state=1)
model.fit(X_train, y_train)

# Mock input for standard execution
current_usage = 19.0

prediction = model.predict([[current_usage]])[0]
label_map = {
    0: "Low Consumption",
    1: "Normal Consumption",
    2: "High Consumption"
}

print("\nClassification Result:")
print("Today's Usage:", current_usage, "units →", label_map[prediction])

if prediction == 2:
    print("\nALERT: HIGH ENERGY CONSUMPTION DETECTED")
    print("Suggestions:\n- Switch off unused appliances\n- Reduce peak hour usage")
elif prediction == 0:
    print("\nINFO: VERY LOW ENERGY CONSUMPTION")
    print("Suggestions:\n- Check for meter or supply issues")
else:
    print("\nSTATUS: ENERGY USAGE IS NORMAL")
    print("Suggestions:\n- Continue monitoring regularly")

print("\nEnergy Monitoring Completed Successfully")

OUTPUT

AI BASED SMART ENERGY MONITORING SYSTEM
Classification Result:
Today's Usage: 19.0 units → High Consumption

ALERT: HIGH ENERGY CONSUMPTION DETECTED
Suggestions:
- Switch off unused appliances
- Reduce peak hour usage
Energy Monitoring Completed Successfully

RESULT

The system successfully detected abnormal energy consumption and generated alerts with corrective suggestions.
