EX NO: 1 - Constructor and Destructor

AIM

To create a class called 'SimpleClass' and demonstrate the invocation of its Constructor and Destructor.

ALGORITHM

  1. Define a class SimpleClass.
  2. Create a Constructor to print an initialization message.
  3. Create a Destructor (~SimpleClass) to print a destruction message.
  4. In main(), create an object. Observe automatic constructor/destructor calls.

PROGRAM

#include <iostream>
using namespace std;

class SimpleClass {
public:
    SimpleClass() {
        cout << "Constructor: Object of SimpleClass created." << endl;
    }
    ~SimpleClass() {
        cout << "Destructor: Object of SimpleClass destroyed." << endl;
    }
    void status() {
        cout << "Message: The class instance is currently in use." << endl;
    }
};

int main() {
    cout << "--- Program Execution Starts ---" << endl;
    SimpleClass obj;
    obj.status();
    cout << "--- Program Execution Ends ---" << endl;
    return 0;
}

OUTPUT

--- Program Execution Starts ---
Constructor: Object of SimpleClass created.
Message: The class instance is currently in use.
--- Program Execution Ends ---
Destructor: Object of SimpleClass destroyed.

RESULT

The program successfully demonstrates that the constructor is called upon object creation, and the destructor is called automatically before termination.

EX NO: 2 - Copy Constructor

AIM

To create a Person class and demonstrate the use of a Copy Constructor to replicate an existing object.

ALGORITHM

  1. Define a Person class with name and age.
  2. Define a Parameterized Constructor to initialize data.
  3. Define a Copy Constructor: Person(const Person &source) to copy members.
  4. In main(), instantiate object 1, then create object 2 as a copy of object 1.

PROGRAM

#include <iostream>
#include <string>
using namespace std;

class Person {
private:
    string name; int age;
public:
    Person(string n, int a) {
        name = n; age = a;
        cout << "Parameterized Constructor called." << endl;
    }
    Person(const Person &source) {
        name = source.name; age = source.age;
        cout << "Copy Constructor called." << endl;
    }
    void display() {
        cout << "Name: " << name << ", Age: " << age << endl;
    }
};

int main() {
    cout << "--- Initializing Original Object ---" << endl;
    Person person1("Alice", 25);
    person1.display();
    
    cout << "\n--- Initializing Copied Object ---" << endl;
    Person person2 = person1;
    person2.display();
    return 0;
}

OUTPUT

--- Initializing Original Object ---
Parameterized Constructor called.
Name: Alice, Age: 25

--- Initializing Copied Object ---
Copy Constructor called.
Name: Alice, Age: 25

RESULT

The copy constructor successfully cloned the data from the source object to the new object.

EX NO: 5 - Polymorphism

AIM

To demonstrate polymorphism using a base class Shape and derived classes Circle and Rectangle to calculate area.

ALGORITHM

  1. Create Base Class Shape with a virtual calculateArea() method.
  2. Create Derived Classes Circle and Rectangle that override the virtual method.
  3. Use a Base Class Pointer (Shape*) to call the area functions at runtime dynamically.

PROGRAM

#include <iostream>
using namespace std;

class Shape {
public:
    virtual void calculateArea() {
        cout << "Area of generic shape is undefined." << endl;
    }
};

class Circle : public Shape {
private: float radius;
public:
    Circle(float r) : radius(r) {}
    void calculateArea() override {
        cout << "Area of Circle: " << 3.14 * radius * radius << endl;
    }
};

class Rectangle : public Shape {
private: float length, width;
public:
    Rectangle(float l, float w) : length(l), width(w) {}
    void calculateArea() override {
        cout << "Area of Rectangle: " << length * width << endl;
    }
};

int main() {
    Shape* shapePtr;
    Circle c(5);
    Rectangle r(4, 6);
    
    cout << "--- Demonstrating Polymorphism ---" << endl;
    shapePtr = &c;
    shapePtr->calculateArea(); 
    
    shapePtr = &r;
    shapePtr->calculateArea(); 
    return 0;
}

OUTPUT

--- Demonstrating Polymorphism ---
Area of Circle: 78.5
Area of Rectangle: 24

RESULT

Polymorphism was successfully demonstrated. The Base pointer dynamically executed the correct overridden method at runtime.

EX NO: 6 - Stack Implementation

AIM

To implement a Stack data structure in C++ using an array-based approach within a class.

ALGORITHM

  1. Define a class Stack with an array and an integer top initialized to -1.
  2. Push: Check for overflow. Increment top and insert the value.
  3. Pop: Check for underflow. Retrieve value and decrement top.

PROGRAM

#include <iostream>
using namespace std;
#define MAX 5

class Stack {
    int top; int arr[MAX];
public:
    Stack() { top = -1; }
    bool push(int x) {
        if (top >= (MAX - 1)) {
            cout << "Stack Overflow!" << endl; return false;
        }
        arr[++top] = x;
        cout << x << " pushed into stack\n";
        return true;
    }
    int pop() {
        if (top < 0) {
            cout << "Stack Underflow!\n"; return 0;
        }
        return arr[top--];
    }
    int peek() {
        if (top < 0) return 0;
        return arr[top];
    }
};

int main() {
    Stack s;
    cout << "--- Stack Operations ---" << endl;
    s.push(10); s.push(20); s.push(30);
    cout << "Top element is: " << s.peek() << endl;
    cout << s.pop() << " popped from stack\n";
    cout << "Top element after pop: " << s.peek() << endl;
    return 0;
}

OUTPUT

--- Stack Operations ---
10 pushed into stack
20 pushed into stack
30 pushed into stack
Top element is: 30
30 popped from stack
Top element after pop: 20

RESULT

The Stack implementation successfully demonstrated LIFO property behavior.

EX NO: 7 - Queue Implementation

AIM

To implement a Queue data structure in C++ using an array-based approach.

ALGORITHM

  1. Define a class Queue with array, front (-1), and rear (-1).
  2. Enqueue: Check if full. Increment rear and store the value.
  3. Dequeue: Check if empty. Retrieve value at front and increment front.

PROGRAM

#include <iostream>
using namespace std;
#define MAX 5

class Queue {
private:
    int items[MAX], front, rear;
public:
    Queue() { front = -1; rear = -1; }
    bool isFull() { return (rear == MAX - 1); }
    bool isEmpty() { return (front == -1 || front > rear); }
    
    void enqueue(int element) {
        if (isFull()) cout << "Queue Overflow!\n";
        else {
            if (front == -1) front = 0;
            items[++rear] = element;
            cout << "Enqueued: " << element << endl;
        }
    }
    int dequeue() {
        if (isEmpty()) return -1;
        int element = items[front++];
        if (front > rear) front = rear = -1;
        return element;
    }
    void display() {
        if (!isEmpty()) {
            cout << "Queue elements: ";
            for (int i = front; i <= rear; i++) cout << items[i] << " ";
            cout << endl;
        }
    }
};

int main() {
    Queue q;
    cout << "--- Queue Operations ---" << endl;
    q.enqueue(10); q.enqueue(20); q.enqueue(30);
    q.display();
    cout << "Dequeued element: " << q.dequeue() << endl;
    cout << "After Dequeue "; q.display();
    return 0;
}

OUTPUT

--- Queue Operations ---
Enqueued: 10
Enqueued: 20
Enqueued: 30
Queue elements: 10 20 30 
Dequeued element: 10
After Dequeue Queue elements: 20 30

RESULT

The Queue program executed successfully, confirming the FIFO property of the data structure.

EX NO: 8 - Expression Evaluation

AIM

To evaluate a given infix mathematical expression using stack-based logic.

ALGORITHM

  1. Create two stacks: values (for integers) and ops (for operators).
  2. Push numbers into values. For operators, pop and apply operations based on precedence rules.
  3. Process parenthesis ( ) recursively by evaluating sub-expressions.
  4. The final result remains in the values stack.

PROGRAM

#include <iostream>
#include <stack>
#include <string>
#include <cctype>
using namespace std;

int precedence(char op) {
    if (op == '+' || op == '-') return 1;
    if (op == '*' || op == '/') return 2;
    return 0;
}
int applyOp(int a, int b, char op) {
    switch (op) {
        case '+': return a + b;
        case '-': return a - b;
        case '*': return a * b;
        case '/': return a / b;
    } return 0;
}

int evaluate(string tokens) {
    stack<int> values; stack<char> ops;
    for (int i = 0; i < tokens.length(); i++) {
        if (isspace(tokens[i])) continue;
        if (isdigit(tokens[i])) {
            int val = 0;
            while (i < tokens.length() && isdigit(tokens[i]))
                val = (val * 10) + (tokens[i++] - '0');
            values.push(val); i--;
        }
        else if (tokens[i] == '(') ops.push(tokens[i]);
        else if (tokens[i] == ')') {
            while (!ops.empty() && ops.top() != '(') {
                int b = values.top(); values.pop();
                int a = values.top(); values.pop();
                char op = ops.top(); ops.pop();
                values.push(applyOp(a, b, op));
            }
            if (!ops.empty()) ops.pop();
        }
        else {
            while (!ops.empty() && precedence(ops.top()) >= precedence(tokens[i])) {
                int b = values.top(); values.pop();
                int a = values.top(); values.pop();
                char op = ops.top(); ops.pop();
                values.push(applyOp(a, b, op));
            }
            ops.push(tokens[i]);
        }
    }
    while (!ops.empty()) {
        int b = values.top(); values.pop();
        int a = values.top(); values.pop();
        char op = ops.top(); ops.pop();
        values.push(applyOp(a, b, op));
    }
    return values.top();
}

int main() {
    string exp = "10+2*6";
    cout << "Expression: " << exp << "\nResult: " << evaluate(exp) << endl;
    return 0;
}

OUTPUT

Expression: 10+2*6
Result: 22

RESULT

The program successfully implements infix expression evaluation respecting BODMAS operator precedence.

EX NO: 9 - Infix to Postfix Notation

AIM

To convert a given infix expression into its equivalent postfix expression using a stack.

ALGORITHM

  1. Scan infix expression. Output operands directly.
  2. Push ( to stack. If ), pop and output until (.
  3. For operators, pop operators from the stack with higher or equal precedence to output, then push the current operator.

PROGRAM

#include <iostream>
#include <stack>
#include <string>
using namespace std;

int prec(char c) {
    if (c == '^') return 3;
    else if (c == '/' || c == '*') return 2;
    else if (c == '+' || c == '-') return 1;
    return -1;
}

void infixToPostfix(string s) {
    stack<char> st; string result;
    for (int i = 0; i < s.length(); i++) {
        char c = s[i];
        if (isalnum(c)) result += c;
        else if (c == '(') st.push('(');
        else if (c == ')') {
            while (st.top() != '(') {
                result += st.top(); st.pop();
            }
            st.pop();
        }
        else {
            while (!st.empty() && prec(s[i]) <= prec(st.top())) {
                result += st.top(); st.pop();
            }
            st.push(c);
        }
    }
    while (!st.empty()) { result += st.top(); st.pop(); }
    cout << "Postfix Expression: " << result << endl;
}

int main() {
    string exp = "A+B*C";
    cout << "Infix Expression: " << exp << endl;
    infixToPostfix(exp);
    return 0;
}

OUTPUT

Infix Expression: A+B*C
Postfix Expression: ABC*+

RESULT

The infix to postfix conversion was successfully executed.

EX NO: 10 - Singly Linked List

AIM

To implement a Singly Linked List and perform Insertion, Deletion, and Traversing.

ALGORITHM

  1. Define a Node structure with data and next pointer.
  2. Insert: Traverse to the end of the list and link a newly created node.
  3. Delete: Search for the node. Bypass it by linking the previous node to its next node.

PROGRAM

#include <iostream>
using namespace std;

struct Node { int data; Node* next; };

class LinkedList {
    Node* head;
public:
    LinkedList() { head = NULL; }
    void insert(int val) {
        Node* newNode = new Node();
        newNode->data = val; newNode->next = NULL;
        if (!head) head = newNode;
        else {
            Node* temp = head;
            while (temp->next) temp = temp->next;
            temp->next = newNode;
        }
    }
    void deleteNode(int key) {
        Node *temp = head, *prev = NULL;
        if (temp && temp->data == key) {
            head = temp->next; delete temp; return;
        }
        while (temp && temp->data != key) {
            prev = temp; temp = temp->next;
        }
        if (!temp) return;
        prev->next = temp->next;
        delete temp;
    }
    void display() {
        Node* temp = head;
        while (temp) { cout << temp->data << " -> "; temp = temp->next; }
        cout << "NULL" << endl;
    }
};

int main() {
    LinkedList list;
    list.insert(10); list.insert(20); list.insert(30);
    list.display();
    list.deleteNode(20);
    list.display();
    return 0;
}

OUTPUT

10 -> 20 -> 30 -> NULL
10 -> 30 -> NULL

RESULT

The Singly Linked List was successfully implemented enabling dynamic memory manipulation.

EX NO: 11 - Doubly Linked List

AIM

To implement a Doubly Linked List and perform Insertion, Deletion, and Forward Traversal.

ALGORITHM

  1. Define a Node with data, next, and prev pointers.
  2. Insert: Traverse to end, link new node, set its prev to the last node.
  3. Delete: Unlink node by updating next of the previous and prev of the next node.

PROGRAM

#include <iostream>
using namespace std;

struct Node { int data; Node* next; Node* prev; };

class DoublyLinkedList {
    Node* head;
public:
    DoublyLinkedList() { head = NULL; }
    void insert(int val) {
        Node* newNode = new Node();
        newNode->data = val; newNode->next = NULL;
        if (!head) { newNode->prev = NULL; head = newNode; }
        else {
            Node* temp = head;
            while (temp->next) temp = temp->next;
            temp->next = newNode; newNode->prev = temp;
        }
    }
    void deleteNode(int key) {
        if (!head) return;
        Node* temp = head;
        while (temp && temp->data != key) temp = temp->next;
        if (!temp) return;
        if (temp == head) head = temp->next;
        if (temp->next) temp->next->prev = temp->prev;
        if (temp->prev) temp->prev->next = temp->next;
        delete temp;
    }
    void displayForward() {
        Node* temp = head;
        while (temp) { cout << temp->data << " <-> "; temp = temp->next; }
        cout << "NULL\n";
    }
};

int main() {
    DoublyLinkedList dll;
    dll.insert(10); dll.insert(20); dll.insert(30);
    dll.displayForward();
    dll.deleteNode(20);
    dll.displayForward();
    return 0;
}

OUTPUT

10 <-> 20 <-> 30 <-> NULL
10 <-> 30 <-> NULL

RESULT

The Doubly Linked List was successfully implemented providing bidirectional data manipulation.

EX NO: 12 - Binary Tree Traversal

AIM

To create a binary tree and perform Preorder, Inorder, and Postorder traversals using recursion.

ALGORITHM

  1. Define a Node structure (left, right, data).
  2. Preorder: Visit Root, then recur left, then recur right.
  3. Inorder: Recur left, then visit Root, then recur right.
  4. Postorder: Recur left, then recur right, then visit Root.

PROGRAM

#include <iostream>
using namespace std;

struct Node {
    int data; Node *left, *right;
    Node(int val) { data = val; left = right = NULL; }
};

void printPreorder(Node* node) {
    if (!node) return;
    cout << node->data << " ";
    printPreorder(node->left); printPreorder(node->right);
}
void printInorder(Node* node) {
    if (!node) return;
    printInorder(node->left);
    cout << node->data << " ";
    printInorder(node->right);
}
void printPostorder(Node* node) {
    if (!node) return;
    printPostorder(node->left); printPostorder(node->right);
    cout << node->data << " ";
}

int main() {
    Node* root = new Node(1);
    root->left = new Node(2); root->right = new Node(3);
    root->left->left = new Node(4); root->left->right = new Node(5);
    
    cout << "Preorder: "; printPreorder(root); cout << endl;
    cout << "Inorder: "; printInorder(root); cout << endl;
    cout << "Postorder: "; printPostorder(root); cout << endl;
    return 0;
}

OUTPUT

Preorder: 1 2 4 5 3 
Inorder: 4 2 5 1 3 
Postorder: 4 5 2 3 1

RESULT

The recursive calls successfully visited the nodes in the correct mathematical order for binary tree traversal.

EX NO: 13 - Binary Search Tree

AIM

To implement a Binary Search Tree and perform insertion and inorder traversal (for sorted output).

ALGORITHM

  1. Define a BST Node structure.
  2. Insert: Recursively compare keys. Insert left if smaller, right if larger.
  3. Inorder: Traverse left, visit root, traverse right to print elements in ascending order.

PROGRAM

#include <iostream>
using namespace std;

struct Node {
    int key; Node *left, *right;
    Node(int item) { key = item; left = right = NULL; }
};

class BinarySearchTree {
public:
    Node* root;
    BinarySearchTree() { root = NULL; }
    
    Node* insert(Node* node, int key) {
        if (!node) return new Node(key);
        if (key < node->key) node->left = insert(node->left, key);
        else if (key > node->key) node->right = insert(node->right, key);
        return node;
    }
    void inorder(Node* root) {
        if (root) {
            inorder(root->left);
            cout << root->key << " ";
            inorder(root->right);
        }
    }
};

int main() {
    BinarySearchTree bst;
    bst.root = bst.insert(bst.root, 50);
    bst.insert(bst.root, 30); bst.insert(bst.root, 20);
    bst.insert(bst.root, 40); bst.insert(bst.root, 70);
    bst.insert(bst.root, 60); bst.insert(bst.root, 80);
    
    cout << "Inorder traversal (Sorted Order): ";
    bst.inorder(bst.root);
    cout << endl;
    return 0;
}

OUTPUT

Inorder traversal (Sorted Order): 20 30 40 50 60 70 80

RESULT

The Binary Search Tree was successfully implemented, confirming elements were sorted following the Left < Root < Right property.
